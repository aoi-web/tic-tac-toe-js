# Tic Tac Toe Game 🎮

A simple Tic Tac Toe game built using HTML, CSS, and JavaScript.

## Features

* Two-player mode (X and O)
* Turn-based gameplay
* Winner detection
* Draw detection
* Reset game functionality

## What I Learned

* DOM manipulation
* Event handling
* Game logic using arrays
* JavaScript loops and functions

## Future Improvements

* Highlight winning boxes
* Better UI design
* Add animations

## How to Run

Open `index.html` in a browser.
